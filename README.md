# 面接用コーディング課題

## 環境構築

まずはクローンをお願いします。

```bash
git clone https://github.com/Akihide-Tsue/apra_coding_front.git
```

起動コマンド

```bash
yarn
yarn dev
```

※ yarnのインストールが必要な場合は

```bash
npm install -g yarn
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## 課題内容

xxxを参照し、APIから取得したデータを使用して同じUIを実装して下さい。  
cssはおおまかな目測で問題ありません。

チェックポイント
